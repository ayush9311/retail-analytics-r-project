# 🛒 Retail Transaction Analytics using R

## 📌 Overview

This project focuses on analyzing retail transaction data to extract meaningful business insights. It includes customer segmentation, sales analysis, and market basket analysis using R.

---

## 🎯 Objectives

* Understand customer purchasing behavior
* Perform sales trend analysis
* Segment customers using RFM analysis
* Discover product associations using Market Basket Analysis

---

## ⚙️ Technologies Used

* R Programming
* RStudio
* tidyverse
* ggplot2
* arules
* arulesViz
* cluster

---

## 📊 Key Analysis Performed

### 🔹 Exploratory Data Analysis (EDA)

* Monthly sales trends
* Top products and customers
* Country-wise revenue analysis

### 🔹 Customer Segmentation

* RFM (Recency, Frequency, Monetary) analysis
* K-Means clustering

### 🔹 Market Basket Analysis

* Association rule mining using Apriori algorithm
* Product recommendation insights

---

## 📈 Results

* Identified high-value customer segments
* Discovered frequently bought product combinations
* Generated actionable insights for business growth
